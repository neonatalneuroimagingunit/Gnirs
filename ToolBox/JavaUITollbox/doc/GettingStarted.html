Getting Started with Widgets Toolbox

What is Widgets Toolbox?

Widgets Toolbox provides a package of MATLAB objects that implement higher level components to use as building blocks for programmatic graphical user interface elements placed within a MATLAB figure window.
Example components are:
  • Superclasses to provide and manage saving and loading session state to and from a file
  • A file selection control, consisting of a label, edit field, and browse button
  • A listbox control combined with a label and a set of buttons for managing the list composition and ordering
  • A tree control providing advanced functionalities like drag and drop, and checkbox nodes

Who should use Widgets Toolbox?

Widgets Toolbox is most applicable to users who already have MATLAB development experience are developing complex applications manually using object-oriented programming. Complex applications may be classified as having a number of different views, or components that require complex interactive functionality. Complex applications may be used by workgroups to make critical design or business decisions, and often require anywhere from a few weeks to many months of development effort.
If you are new to MATLAB or to building MATLAB applications, or if you have a simple application, it is recommended you first review MATLAB's new App Designer tool and review the documentation on Differences Between App Designer and GUIDE.

Layout Management

An important aspect of a complex application is layout management. It is recommended that you consider also using GUI Layout Toolbox as a companion to Widgets Toolbox. GUI Layout Toolbox provides a framework to manage positioning and resizing of complex user interface layouts.

Limitations

These components are intended to be used with or added to a traditional MATLAB figure programmatically. However, they may also be added programmatically to a GUIDE-generated app, but not within the graphical GUIDE editor. The components do not currently support AppDesigner or uifigure containers.

Installation

Widgets Toolbox is provided as a MATLAB toolbox file (.mltbx).
For instructions on installing and uninstalling, see the section on managing toolboxes in the MATLAB documentation.
If you like this toolbox, please help others to find it by leaving a rating and comment on the MATLAB Central File Exchange.
